# 术语表

Script Graph 文档中使用的核心术语说明。docsify-glossary 插件会根据本页的术语，在其它页面中自动替换为指向这里的链接。

#### 脚本图
Script Graph 编辑器中，通过节点和连线可视化编写 TypeScript 代码的图表。

#### Script Graph
与“脚本图”同义，用于指代本插件提供的脚本图编辑器。

#### 工作流
由多个节点和连线组合而成的执行流程，用于描述一段完整的游戏逻辑或功能。

#### AST 模式
基于 TypeScript 抽象语法树直接编辑代码的模式，适合熟悉代码结构的高级用户。

#### 节点
工作流中的基础执行单元，包含输入、输出端口和属性，可以代表一次函数调用或一个逻辑步骤。

#### 边
连接两个节点的连线，表示数据流或执行顺序。

#### 变量
在脚本图中可复用的数据单元，可以在不同节点之间读写，用于存储和传递状态。
