# Script Graph

> 一款专为 Cocos Creator 设计的可视化脚本编辑插件

## 简介

Script Graph（脚本图编辑器）是一款基于节点连线的可视化脚本编辑工具，让零基础用户也能快速高效地完成游戏脚本编写。

## 核心特性

-   🎨 **可视化编辑** - 通过节点连线方式编辑代码
-   � **双模式支持** - AST 模式 + 工作流模式
-   💡 **零基础友好** - 无需编程基础，快速上手
-   ⚡ **高性能** - 编译为高性能的 TypeScript 代码
-   🎯 **丰富节点** - 120+ 种内置节点类型
